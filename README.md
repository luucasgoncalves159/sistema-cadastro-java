# 🧾 Sistema de Cadastro de Produtos

Projeto desenvolvido em Java com o objetivo de praticar conceitos de programação orientada a objetos (POO) e lógica de programação.

---

## 🚀 Funcionalidades

- Cadastro de produtos
- Exibição de produtos cadastrados
- Estrutura baseada em classes

---

## 🛠️ Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)

---

## 📁 Estrutura do projeto

- `Principal.java` → Classe principal com o método `main`
- `Produto.java` → Classe que representa o produto
